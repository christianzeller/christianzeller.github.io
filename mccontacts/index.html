Help: Meshcore Export / Import

Export contacts from Meshcore

  1. Open your Meshcore app.
  2. Go to Settings (or equivalent configuration menu).
  3. Select Export Config.
  4. Choose to export only contacts (no other settings).
  5. Save the exported .json file to your device.

Import cleaned contacts into Meshcore

  1. Keep your original export file as a backup.
  2. Use this tool to clean up and download the cleaned contacts JSON.
  3. In Meshcore, go to Settings → Import Config.
  4. Select the cleaned contacts JSON file.
  5. Confirm the import and verify that your contacts look correct.

Recommendation: Always keep the original export file in case something goes wrong and you need to restore your previous contacts.
